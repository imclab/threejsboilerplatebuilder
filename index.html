About

This is a builder for a boilerplate for three.js. You can find more doc here. It has been written by Jerome Etienne. It uses jszip , a library which create .zip files with Javascript. Its creates the boilerplater.zip that you download. It uses downloadify , a small library to create and download files without server. Thus i don't have to run servers, and this makes me happy :) Last but not least, it uses three.js, the javaScript 3d engine that we all love :) Boilerplate Builder for Three.js is on github under MIT license.
